# 故障排查

## 模组没有“选项”按钮

- 确认 `ONITalk.dll`、`mod.yaml`、`mod_info.yaml` 位于同一个 ONITalk 文件夹。
- 从本地 DLL 更新后完全重启游戏。
- 查看 `Player.log` 中是否出现 `Mod ONITalk initialized`。

## 没有台词

- 确认“启用 ONITalk”已勾选。
- 测试时可启用高频模式并缩短测试间隔。
- 睡觉、如厕等状态会被过滤；复制人还需要满足距离和会话冷却条件。
- 远程接口不可用时会自动使用离线台词。

## API 验证失败

- 用服务商预设重新填充 Endpoint 和模型。
- 确认 Key 属于所选服务商，并检查账户余额或配额。
- 401/403 通常是鉴权问题；429 是频率或配额限制；模型不存在时检查模型名称。
- 连续失败会触发最长五分钟退避，避免反复请求；成功请求后会自动恢复。

## 聊天窗口不见了

- 在选项中确认“显示聊天历史窗口”已启用。
- 窗口位置按屏幕比例保存；更换分辨率后可临时删除配置中的 `chatWindowX` 和 `chatWindowY`，或备份后重置配置。
- 标题栏的 `—` 可最小化，`+` 可恢复。

## 语言没有更新

- 在 ONITalk 选项顶部选择语言并保存。
- 台词语言自动跟随最终应用的 ONITalk 界面语言。
- 若旧窗口没有刷新，请关闭并重新打开选项；极少数游戏 UI 缓存需要重启。

## 日志位置

```text
%USERPROFILE%\AppData\LocalLow\Klei\Oxygen Not Included\Player.log
```

公开日志前请删除 API Key、个人路径和不相关内容。API Key 正常情况下不会由 ONITalk 写入日志。
