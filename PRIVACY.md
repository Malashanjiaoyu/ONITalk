# 隐私与网络请求

ONITalk 是本地运行的《缺氧》模组。它不会运营独立服务器，也不会收集遥测。

## 会保存在本机的内容

- API 服务商、Endpoint、模型和 API Key；
- 窗口位置、语言和生成选项；
- 按殖民地隔离的近期对话、行动与重大事件记忆；
- 玩家主动导出的记忆 JSON。

API Key 以明文保存在 ONITalk 配置文件中，这是游戏模组配置的限制。不要上传配置文件、完整 Player.log 或记忆导出。

## 会发送给所选 AI 服务商的内容

使用远程接口时，每次请求可能包含：

- 参与对话的复制人姓名、特质、当前工作和环境状态；
- 当前触发原因、上一句台词；
- 本地检索选中的近期关系、行动和殖民地事件摘要；
- 玩家配置的自定义风格提示词。

请求直接发送到玩家配置的 Endpoint。数据处理、保留和费用遵循对应服务商的条款。选择“离线台词”时不会发送远程请求。

## 日志与导出

ONITalk 不会把 API Key写入日志。日志可能包含生成台词、复制人名称、殖民地名称和可读错误。记忆导出不包含 API Key，但包含游戏内生成内容。

提交问题前，请只复制必要日志片段，并再次检查其中没有私人信息或密钥。
