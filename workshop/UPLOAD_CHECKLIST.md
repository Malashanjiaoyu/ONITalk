# Steam 创意工坊上传与验收清单

## 上传前

- [ ] GitHub v0.10.1 预发行 ZIP 已在干净目录解压并加载。
- [ ] 模组文件不含 `ONITalk.json`、memory、exports、Player.log 或 API Key。
- [ ] `mod_info.yaml` 版本与详情页一致。
- [ ] 基础游戏和 Spaced Out! 均能加载。
- [ ] 中文、英语、西班牙语界面至少各打开一次。
- [ ] 主预览图、6 张截图和 BBCode 详情页已经终审。
- [ ] 截图中的 API Key 字段为空或完全遮挡。

## 首次上传

- [ ] 初始可见性设为 Hidden。
- [ ] 标题、简介、标签、依赖和兼容内容正确。
- [ ] 上传内容来自 GitHub Release ZIP 解压结果，而不是带本地配置的游戏目录。
- [ ] 上传后从创意工坊页面重新订阅到干净目录。

## 隐藏状态验收

- [ ] 订阅下载后文件结构正确。
- [ ] 主菜单出现 ONITalk 和“选项”按钮。
- [ ] 离线模式无需 Key 即可产生台词。
- [ ] 远程验证只发送一次最小请求。
- [ ] 聊天窗口、记忆预览和记忆库正常。
- [ ] 卸载后不影响存档；配置与记忆是否保留符合文案说明。

## 公开前

- [ ] GitHub、Steam 版本号和更新日志一致。
- [ ] 费用、隐私、第三方服务条款和非 Klei 官方声明清晰可见。
- [ ] 至少一位非开发环境用户完成安装。
- [ ] 已准备置顶评论：反馈格式、日志位置和禁止公开 API Key。
