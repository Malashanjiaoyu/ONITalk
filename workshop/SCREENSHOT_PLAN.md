# Steam 创意工坊截图计划

目标：6 张真实游戏截图，统一使用 1920×1080、100% UI 缩放、无调试控制台、无 API Key。截图前将测试提示词从 `123` 恢复为正常内容。

## 1. 主功能图

- 场景：殖民地正常工作，多名复制人在画面中。
- 展示：两名复制人头顶台词与半透明 ONITalk 对话窗口。
- 说明：`Context-aware Duplicant conversations / 情境感知复制人对话`。

## 2. 连续对话

- 展示：聊天窗口内同一对复制人的两句自然回应。
- 要求：内容不重复、不含测试后缀，能看出上下句关联。

## 3. 记忆注入预览

- 点击标题栏“忆”。
- 展示：关系、行动、殖民地事件、评分和 Token 预算。
- 避免出现私人殖民地名称时，可使用专门测试存档。

## 4. 长期记忆库

- 点击标题栏“库”。
- 展示：关系、行动和事件三类摘要，以及刷新／导出／清空按钮。
- 不展示导出的本地绝对路径。

## 5. 服务商配置

- 展示：服务商下拉框、Endpoint、模型和“验证配置”。
- API Key 字段必须为空或完全遮挡；截图后再次检查像素中没有密钥。

## 6. 多语言与窗口布局

- 英语或西班牙语界面；聊天窗口放在不遮挡关键 UI 的位置。
- 展示语言切换与窗口透明度，证明不是仅中文界面。

## 拍摄检查

- 关闭 Steam、Discord、性能监控等覆盖层通知。
- 不出现现实姓名、桌面路径、账号、API Key 或聊天通知。
- 保留一份无文字标注原图；创意工坊上传图可另存压缩版。
- PNG 作为母版，上传前再按平台限制导出 JPG/PNG。
